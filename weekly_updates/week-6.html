Concepts Overview

TSS — True Skill Statistic

Measures: “How well can the model separate positives from negatives?”
Range: -1 to +1

  • -1 → Worse than random guessing
  • 0 → Equivalent to random guessing
  • +1 → Perfect skill
TSS is not sensitive to class imbalance, making it better than accuracy for skewed datasets.

HSS — Heidke Skill Score

Measures: “How much better is this model than random guessing?”
Compares your model's accuracy against what a random classifier would achieve.

$$ TSS = \frac{TP}{TP + FN} - \frac{FP}{FP + TN} $$

(Forecast is better than random one)

$$ HSS = \frac{2 \times (TP \times TN - FN \times FP)} {(P \times (FN + TN) + (TP + FP) \times N)} $$

Dynamic Time Warping (DTW)

DTW is a distance-based algorithm used for time-series similarity. It allows sequences to be “stretched” or “compressed” so that similar patterns align even if they occur at different speeds.

DTW Algorithm (Pseudocode)

Given two sequences X = (x1, x2, ..., xn)
                    Y = (y1, y2, ..., ym)

Create an n × m cost matrix `dp`
Initialize dp with ∞
dp[0][0] = 0

For i from 1 to n:
    For j from 1 to m:
        cost = |xi − yj|
        dp[i][j] = cost + min(
                          dp[i-1][j],      # insertion
                          dp[i][j-1],      # deletion
                          dp[i-1][j-1]     # match
                       )

DTW distance = dp[n][m]

This algorithm computes the minimum cumulative distance needed to warp two sequences into alignment.

Distance Metric Interpretation

DTW gives a distance score:

  • Lower distance → more similar time-series
  • Higher distance → more different
